# 🎬 CineStar Booking System (FastAPI)

An authentic, adaptive movie booking API built sequentially over a 6-day development roadmap. This project covers everything from basic CRUD operations to advanced search, filtering, and a multi-step seat-hold workflow.

---

## 🚀 Quick Start

### 1. Installation

Ensure you have Python 3.8+ installed. Install dependencies using:

```bash
pip install -r requirements.txt
```
